tools

Disclaimer: I may be not using one of these tools, because i change tools every time, but i use some of them for training purposes.

Programming Stuff

  • Visual Studio

    It's the IDE that i use to develop the game itself and i use the programming language C# for all the programming in the game.

  • Visual Studio Code

    It's the IDE that i use to develop websites / bots for discord and etc.

  • Arduino

    It wouldn't be necessary for me to put this here, but I use Arduino with the C language to create my random inventions and with its microcontroller.

  • Monogame

    Monogame is the C# Framework based in XNA that its used for game development and features a lot of custom-tools.

  • Monogame.Extend

    Monogame.Extend is bassicaly the set of utility classes and extension methods for MonoGame that provides a wide range of features for game development. We can say that has all the tools for game development until the tools that a game engine have itself.

  • Unity

    Unity is one of the most popular free and best game engines to make a full game with agility and precision. I use Unity to make a few games, since i don't have used Unity a long time i would recommend for who wants to make games, use it first before using a framework (if you want tho).

Art / Design

  • Aseprite

    Aseprite is the software i use to make all the pixel art and animations. Currently its paid, but if you want something like Aseprite i would recommend check out the software "Libresprite".

  • Tiled

    Tiled is the map editor which i use to make all the levels in the game, i use a different version of Tiled which uses C#.

  • Blender

    Blender is the free and open source 3D / 2D creation suite. It supports the entirety of the 3D / 2D pipeline—modeling, rigging, animation, simulation, rendering and etc. I use this as for VFX / animation stuff

  • Animation Desk

    For me the best 2D animation software, its free and available for Windows / IOS / Android for free. I use to make some animations.

  • Sketchbook

    Sketchbook is a free/paid software made for illustration/art. If you use android you may know this app as the most recommended app for drawing. Sometimes i use to train my drawing skills, also i use the free version for mobile with my Tab A7 Lite with Stylus Pen.

Audio / Music

  • Ableton Live 11 Lite

    Ableton Live 11 Lite is a free version of the DAW (Digital Audio Workstation) Ableton Live 11 that i use to make all the soundtracks.

  • Audacity

    Audacity is a free and open-source digital audio editor. I use to make all the adjustments in the soundtrack / audio of the game.

  • FMOD Studio

    FMOD Studio is the tool that i use to design all the audio in the game.

  • LMMS

    LMMS is a free DAW which i use as an alternative to Ableton Live 11, because its have his limitations.
